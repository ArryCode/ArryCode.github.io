第二章 控制流

与其他编程语言一样，Python的流程控制，也是通过三种语句实现。

2.1 布尔值、比较操作符和布尔操作符

在学习控制语句之前，我们首先要知道在python中如何表达yes或no，所以我们需要认识布尔值、比较操作符和布尔操作符。

布尔变量只有True和False两种取值结果。需要注意大小写。

比较操作符：== != < > >= <=

布尔操作符：and or not

值得一提的是，not操作符，可以嵌套。在Python中，布尔操作符的操作顺序是：not and or

在用于条件时，0,0.0，‘ ’会被认为是 False，其他值被认为是 True

2.4 for循环和range()函数

执行固定的次数，首选for循环。

i in range(a)：$i$ 取值从 $0$ 到 $1-1​$，每次递增1

i in range(a,b): $i$ 取值从$a$到$b-1$，每次递增1

i in range(a,b,c): $i$取值$a$到$b-1$，每次递增$c$

2.5 break语句和continue语句

break提前结束整个循环。也就是碰到break语句直接结束循环。

continue语句提前进入下一次循环，也就是回到循环开始的地方。

这两个语句，一般情况下会与 if 语句结合使用。
